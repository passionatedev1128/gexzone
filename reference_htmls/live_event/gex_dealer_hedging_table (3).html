⬡ Anexo — Por qué el Dealer realiza esa acción después de la expiración
01 Inversor compra Calls — Dealer Short Calls
ITM
El inversor ejerce y el Dealer debe entregar acciones. Para cerrar su posición corta, el Dealer compra (BUY) en el mercado.
OTM
No hay ejercicio. El Dealer vende (SELL) las acciones que tenía como cobertura, cerrando la posición residual.
02 Inversor vende Calls — Dealer Long Calls
ITM
El Dealer es ejercido y debe entregar acciones. Por eso vende (SELL) las acciones en el mercado para cumplir la asignación.
OTM
No hay ejercicio. El Dealer compra (BUY) las acciones que tenía como cobertura para cerrar la posición larga residual.
03 Inversor compra Puts — Dealer Short Puts
ITM
El inversor ejerce y obliga al Dealer a comprar las acciones. El Dealer compra (BUY) en el mercado para cumplir con la asignación.
OTM
No hay ejercicio. El Dealer vende (SELL) las acciones que tenía como cobertura para cerrar la posición.
04 Inversor vende Puts — Dealer Long Puts
ITM
El Dealer es asignado y debe comprar acciones. Para cerrar la posición, el Dealer vende (SELL) las acciones en el mercado.
OTM
No hay asignación. El Dealer compra (BUY) las acciones que tenía como cobertura para cerrar la posición larga residual.
